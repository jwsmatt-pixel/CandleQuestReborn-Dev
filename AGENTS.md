# Candle Quest Codex Instructions

Candle Quest is a mobile-first candlestick recognition game.

Current priority:
- Keep the game lightweight, fast, and easy to test.
- Mobile layout and summary screen polish matter more than new features.
- Do not rewrite the whole app unless explicitly asked.
- Prefer small, safe patches.

Core files:
- index.html
- style.css
- game.js
- README.md
- CHANGELOG.md
- CHECKPOINT_SUMMARY.md

Rules:
- Preserve existing game modes unless directly asked to change them.
- Keep homepage and summary screen logic separate.
- Update CHANGELOG.md and CHECKPOINT_SUMMARY.md when behaviour changes.
- After edits, explain changed files, test steps, risks, and assumptions.
